# 📊 Pharmacovigilance EDA Analysis

> **Exploratory Data Analysis (EDA) and Statistical Analysis of a Synthetic Pharmacovigilance Dataset using Python**

This repository demonstrates an end-to-end Exploratory Data Analysis (EDA) workflow on a synthetic Pharmacovigilance dataset. The project focuses on understanding data quality, identifying trends, performing statistical analysis, and generating meaningful insights using Python.

> **Disclaimer:** This project uses a **synthetic (dummy) dataset** created solely for educational and portfolio purposes. It does **not** contain any real patient, clinical, or confidential information.

---

# 📌 Project Objectives

- Perform comprehensive Exploratory Data Analysis (EDA)
- Clean and preprocess the dataset
- Assess data quality and consistency
- Identify missing values and duplicate records
- Detect and analyze outliers
- Explore feature distributions
- Analyze relationships between variables
- Perform descriptive and inferential statistical analysis
- Generate meaningful visualizations
- Derive actionable insights from the data
